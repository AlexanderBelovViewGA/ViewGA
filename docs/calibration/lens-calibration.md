# Калибровка линзы (CGF ViewGA Lens Calibrator)

Для настройки оптики используется специализированное приложение **CGF ViewGA Lens Calibrator**. Интерфейс построен на единой платформе с основной системой, что упрощает навигацию.

## 1. Подключение и подготовка
Перед началом работы необходимо убедиться в стабильности входящих сигналов:

*   **Видеоисточник**: Настройка подключения к видеопотоку (например, через платы захвата Blackmagic).
*   **Трекер**: Подключение к Power Box (обычно осуществляется через COM-порт).
*   **Энкодеры**: В блоке Lens Trackers должен отображаться статус «Подключено». Если энкодеры не калиброваны, система потребует провести их инициализацию.
*   **Визуализация**: Для наглядности рекомендуется вывести на экран калибровочный паттерн (опционально).

## 2. Параметры камеры и сенсора
Система позволяет сохранять историю калибровок для одной линзы, разделяя их по датам внутри одного профиля.

### Характеристики сенсора:
1.  **Формат съемки**: Укажите режим работы сенсора. 
    *   Для кинокамер: `Open Gate` или `Full Frame`.
    *   Для ТВ-стандартов: например, `4/3`.
2.  **Физический размер**: Укажите размер рабочей области матрицы (например, `8.8 мм`).
3.  **Разрешение**: Выберите формат потокового изображения (например, `Full HD`).
4.  **Кроп-фактор**: 
    *   Если используется не весь сенсор, укажите параметры кропа. 
    *   *Примечание: Текущая версия поддерживает ввод кропа в пикселях для максимальной точности, планируется добавление коэффициента кроп-фактора.*

## 3. Описание линзы
В соответствующие поля занесите метаданные:
*   Тип и название линзы.
*   Серийный номер.
*   Пользовательские комментарии (условия съемки, специфика сетапа).

## 4. Маппинг энкодеров
Уникальная особенность ViewGA — привязка к реальным физическим меткам («рисочкам») на объективе, а не к абстрактным процентам работы энкодера.

**Процесс маппинга:**
1. Выберите конкретную метку на кольце объектива (фокус или зум).
2. Присвойте ей имя в программе, соответствующее значению на линзе.
3. Зафиксируйте положение энкодера для этой метки.
4. Повторите процедуру для всех ключевых положений.

!!! tip "Преимущество метода"
    Такой подход позволяет оператору и инженеру говорить на одном языке, оперируя реальными значениями дистанции или фокусного расстояния, а не техническими данными сенсора.
